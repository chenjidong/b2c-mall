### 环境配置
nodejs 8.5+
npm 6.13+

### npm 依赖安装失败问题 使用淘宝镜像
* npm install -g cnpm --registry=https://registry.npm.taobao.org
* cnpm install
* cnpm run dev

#### 安装教程

1. 执行 npm install
2. debug npm run dev

### 打包
* 域名修改在 config 目录下 对应各个环境
* npm run build:prod
